# OBS Music TV para GitHub Pages

## Archivos

- player.html: reproductor limpio para OBS.
- admin.html: panel de control.
- config.json: lista de música, anuncios y configuración.
- music/: videos musicales.
- ads/: anuncios.
- assets/: logos o recursos.

## Uso en GitHub Pages

1. Sube este contenido a tu repositorio.
2. Sube tus videos MP4 a music/ y ads/.
3. Edita config.json.
4. En OBS usa player.html como Fuente Navegador.
5. Abre admin.html en tu navegador para controlar.

Ejemplo:
https://usuario.github.io/repositorio/player.html
https://usuario.github.io/repositorio/admin.html
